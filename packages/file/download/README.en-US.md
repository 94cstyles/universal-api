## Example

```js
import file from 'universal-file';

file.download({
  url: 'http://img.alicdn.com/tfs/TB1x669SXXXXXbdaFXXXXXXXXXX-520-280.jpg',
  success: res => {
    console.log(res.filePath);
  },
  fail: res => {
    console.log(res);
  },
});

```

You can also import from the big package:

```js
import { file } from 'universal-api';

```
