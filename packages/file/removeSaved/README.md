## 方法

删除某个保存的文件。

## 示例

```js
import file from 'universal-file';

file.removeSaved({
  filePath: '**filePath**',
  success:(res) => {
    console.log('remove success');
  }
});

```

你也可以从大包引入：
```js
import { file } from 'universal-api';

```
