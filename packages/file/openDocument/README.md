## 示例

```js
import file from 'universal-file';

file.openDocument({
  filePath: '**filePath**',
  fileType: 'pdf',
  success: (res) => {
    console.log('open document success');
  };
});

```

你也可以从大包引入：
```js
import { file } from 'universal-api';

```
