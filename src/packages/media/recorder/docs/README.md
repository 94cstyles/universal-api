## 安装

```bash
$ npm install @uni/recorder --save
```
or
```bash
$ npm install @uni/apis --save
```
## 示例

```javascript
import getRecorderManager from '@uni/recorder';

const recorderManager = getRecorderManager();

```

你也可以从大包引入：
```js
import { getRecorderManager } from '@uni/apis';
```
